#### コマンド  
以下のコマンドでビルドが出来ます。  
初回のビルドには数分かかります。  
```
git clone https://github.com/blz-soft/CryptoTool.git
cd CryptoTool
cargo build --release
```
`--release`を付けないとリリースビルドになりません。  
デバッグビルドですと、処理時間が非常に長くなります。
